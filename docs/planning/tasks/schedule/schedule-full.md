## デモシナリオ（更新版）

### 基本デモ（3分）

```
1. アプリ起動 → オンボーディング（権限許可）
2. Sign in with Apple でログイン
3. お気に入り曲を設定（Spotify 検索 → 選択）
4. 別の iPhone でも同様にセットアップ
5. 2台を近づける → BLE ですれ違い発生
6. プッシュ通知が届く
7. すれ違い履歴を開く → 相手の曲が表示される
8. 歌詞入力モーダルで一言投稿「今日も空は青かった」
```

### 拡張デモ（時間があれば +2分）

```
9. 「あと○人で曲が生まれます」の表示
10. 事前に生成しておいた楽曲を再生
    → 「4人の出会いから生まれた曲」
11. 歌詞一覧と参加者が表示される
```

## 備考

- **Lyria 連携**: 本番 API が使えない場合はモッククライアントを使用し、デモでは事前生成した楽曲を再生
- **Gemini 連携**: 歌詞分析が間に合わない場合は固定パラメータ（mood: "upbeat", genre: "J-POP"）で代替
- **Android**: 今回のスコープ外。iOS 優先で実装し、後日粉川が Android 版を実装
