## デモシナリオ（更新版）

### 基本デモ（3分・スコープ調整版）

```
1. アプリ起動 → BLE権限許可
2. 2台の iPhone を用意
3. 両方でアプリ起動・BLE開始
4. 2台を近づける → BLE ですれ違い発生
5. すれ違い通知が表示される
6. すれ違い履歴を開く → すれ違い相手が表示される
7. 歌詞入力モーダルで一言投稿「今日も空は青かった」
8. 他のユーザーも投稿 → 一定数で楽曲生成開始
```

### 拡張デモ（時間があれば +2分）

```
9. 「あと○人で曲が生まれます」の表示
10. 事前に生成しておいた楽曲を再生
    → 「4人の出会いから生まれた曲」
11. 歌詞一覧と参加者が表示される
```

## 備考

- **Lyria 連携**: 本番 API が使えない場合はモッククライアントを使用し、デモでは事前生成した楽曲を再生
- **Gemini 連携**: 歌詞分析が間に合わない場合は固定パラメータ（mood: "upbeat", genre: "J-POP"）で代替
- **Android**: 今回のスコープ外。iOS 優先で実装し、後日粉川が Android 版を実装
