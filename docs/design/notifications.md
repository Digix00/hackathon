# 通知デザイン

## 概要

プッシュ通知、アプリ内通知の全パターンを定義する。通知は控えめに、しかし必要な情報は確実に届ける。

---

## 設計原則

| 原則 | 適用 |
|---|---|
| バッチ処理 | 短時間の連続すれ違いはまとめて通知 |
| 即時性 | 重要な通知は即座に |
| 文脈 | 何が起きたか、次に何ができるかを明示 |
| 控えめ | 過度な通知でユーザーを疲れさせない |

# 1. プッシュ通知

## 1.1 すれ違い通知

### 単一すれ違い

```
┌─────────────────────────────────────┐
│ 🎵 すれ違い                    今   │
│                                     │
│ 新しい音楽と出会いました             │
│ 「曲名」- アーティスト名             │
└─────────────────────────────────────┘
```

**iOS リッチ通知**:
```
┌─────────────────────────────────────┐
│ 🎵 すれ違い                    今   │
│                                     │
│ ┌────┐ 新しい音楽と出会いました      │
│ │ 🎵 │ 「夜に駆ける」              │
│ └────┘ YOASOBI                     │
│                                     │
│ [▶ 再生]           [♡ いいね]      │
└─────────────────────────────────────┘
```

**アクション**:
- 通知タップ → すれ違い詳細画面
- 「再生」→ 音楽アプリで再生
- 「いいね」→ いいね追加、アプリは開かない

### 複数すれ違い（バッチ）

5分以内に複数すれ違いがあった場合。

```
┌─────────────────────────────────────┐
│ 🎵 すれ違い                   5分前 │
│                                     │
│ 3人と新しい音楽に出会いました        │
│ 「曲名A」「曲名B」など              │
└─────────────────────────────────────┘
```

**タップ**→ すれ違い履歴画面

### 通知タイミング

| 設定 | タイミング |
|---|---|
| 即時 | すれ違い成立から5秒以内 |
| 1時間ごと | 毎時0分にまとめて |
| 1日1回 | 20:00にまとめて |

## 1.4 システム通知

### 権限リマインダー

Bluetoothや位置情報がオフの場合（1日1回まで）。

```
┌─────────────────────────────────────┐
│ 📍 すれ違いアプリ             12:00 │
│                                     │
│ 位置情報がオフになっています         │
│ すれ違いの精度を上げるにはオンに     │
└─────────────────────────────────────┘
```

### 久しぶりリマインダー

3日以上アプリを開いていない場合（週1回まで）。

```
┌─────────────────────────────────────┐
│ 🎵 すれ違いアプリ             15:00 │
│                                     │
│ 最近すれ違いがありました             │
│ どんな音楽と出会ったか見てみよう     │
└─────────────────────────────────────┘
```

# 2. アプリ内通知

## 2.1 トースト通知

画面上部から一時的に表示。

### すれ違い成立（アプリ起動中）

```
┌─────────────────────────────────────┐
│ 🎵 新しいすれ違い！                  │
│    「曲名」- アーティスト     [見る] │
└─────────────────────────────────────┘
```

**仕様**:
- 表示位置: Safe Area 下、16pt マージン
- 表示時間: 4秒（タップで即消え）
- アニメーション: 上からスライドイン
- 「見る」タップ: 詳細画面へ
- トースト外タップ: 消える（履歴には残る）

### アクション完了

```
┌─────────────────────────────┐
│ ✓ いいねしました            │
└─────────────────────────────┘

┌─────────────────────────────┐
│ ✓ ブロックしました          │
└─────────────────────────────┘

┌─────────────────────────────┐
│ ✓ 通報を受け付けました       │
└─────────────────────────────┘
```

**仕様**:
- 表示時間: 2秒
- 背景: Slate 800（ライト）/ Slate 200（ダーク）
- テキスト: 白（ライト）/ Slate 900（ダーク）

### エラー

```
┌─────────────────────────────┐
│ ⚠️ 接続できませんでした      │
└─────────────────────────────┘
```

**仕様**:
- 背景: Red 600
- テキスト: 白
- 表示時間: 3秒

## 2.2 バナー通知

画面上部に固定表示（消すまで残る）。

### 権限不足

```
┌─────────────────────────────────────┐
│ ⚠️ Bluetoothがオフです             │
│    すれ違いを検知できません   [設定] │
└─────────────────────────────────────┘
```

### オフライン

```
┌─────────────────────────────────────┐
│ 📡 オフラインです                   │
│    一部機能が制限されています        │
└─────────────────────────────────────┘
```

**仕様**:
- 位置: Safe Area 下、固定
- 高さ: auto（内容に応じて）
- 背景: Yellow 50（警告）/ Red 50（エラー）
- 閉じるボタン: なし（状況解消で自動消去）

# 3. 通知センター（アプリ内）

## 3.1 通知一覧画面

ナビゲーションバーの🔔タップで表示。

```
┌─────────────────────────────┐
│ ←        通知               │
├─────────────────────────────┤
│                             │
│  今日                       │
│  ┌───────────────────────┐  │
│  │ 🎵 新しいすれ違い      │  │
│  │    「曲名」- アーティスト│  │
│  │                  10分前 │  │
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
│  │ 💬 コメント            │  │
│  │    ユーザー名さんが... │  │
│  │                  1時間前│  │
│  └───────────────────────┘  │
│                             │
│  昨日                       │
│  ┌───────────────────────┐  │
│  │ 🎵 3人とすれ違いました  │  │
│  │                  昨日   │  │
│  └───────────────────────┘  │
│                             │
└─────────────────────────────┘
```

## 3.2 通知アイテム

```
高さ: 72pt
パディング: 16pt
未読: 左端に青ドット（6pt）
既読: ドットなし
タップ: 該当画面へ遷移
```

## 3.3 未読バッジ

```
ナビゲーションバーの🔔右上に:
- 未読1-9件: 数字バッジ
- 未読10件以上: 「9+」
- 未読なし: バッジなし

バッジスタイル:
- サイズ: 16pt × 16pt（min）
- 背景: Red 500
- テキスト: 白、10pt Bold
```

# 5. 通知文言一覧

## すれ違い

| パターン | タイトル | 本文 |
|---|---|---|
| 単一 | すれ違い | 新しい音楽と出会いました「曲名」- アーティスト |
| 複数 | すれ違い | N人と新しい音楽に出会いました |
| 初回 | はじめてのすれ違い | おめでとう！「曲名」と出会いました |
| 再会 | また会えた！ | ユーザー名さんと再びすれ違いました |

## コメント

| パターン | タイトル | 本文 |
|---|---|---|
| 新規 | コメント | ユーザー名さんがコメントしました「内容...」 |
| 返信 | 返信 | ユーザー名さんが返信しました「内容...」 |

## システム

| パターン | タイトル | 本文 |
|---|---|---|
| BT OFF | すれ違いアプリ | Bluetoothがオフです。すれ違いを検知できません |
| 位置OFF | すれ違いアプリ | 位置情報がオフです。精度を上げるにはオンに |
| 久しぶり | すれ違いアプリ | 最近すれ違いがありました。どんな音楽と出会ったか見てみよう |

# 6. ハプティクス・サウンド

## プッシュ通知

- **サウンド**: システムデフォルト（カスタム音は将来対応）
- **バッジ**: 未読数を表示

## アプリ内トースト

- **ハプティクス**: `.notification(.success)`（成功時）
- **サウンド**: なし
