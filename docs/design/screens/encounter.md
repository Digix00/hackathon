# すれ違い履歴・詳細画面 UI設計

## 概要

すれ違いの履歴一覧と、個別の詳細画面を設計。曲情報の閲覧、いいね、コメント、通報などのインタラクションを提供する。

---

## 画面一覧

1. **履歴一覧画面** - すれ違いの時系列リスト
2. **詳細画面** - 個別のすれ違い詳細とコメント

---

## セクション詳細

### ナビゲーションバー
- 左: 戻るボタン（ホームから遷移時）またはタイトルのみ（タブから）
- 中央: 「すれ違い履歴」
- 右: 検索アイコン

### 日付セクションヘッダー

**表示ルール**:
| 日付 | 表示 |
|---|---|
| 当日 | 今日 |
| 前日 | 昨日 |
| 今週内 | 曜日（月曜日など） |
| それ以前 | 日付（3月12日） |

**スタイル**:
```
フォント: 13pt Semibold
色: Slate 500
マージン: 上24pt、下8pt
背景: sticky（スクロール時固定）
```

### リストアイテム

```
┌─────────────────────────────────────────┐
│  ┌────┐                                │
│  │    │  曲名                 ユーザー名 │
│  │ 🎵 │  アーティスト名                 │
│  │    │                        10:30   │
│  └────┘                                │
└─────────────────────────────────────────┘
```

**スタイル**:
```
高さ: 80pt
パディング: 16pt
アルバムアート: 48pt × 48pt、角丸 6pt
曲名: 15pt Semibold
アーティスト: 13pt Regular、Slate 500
ユーザー名: 13pt Regular、右寄せ
時刻: 13pt Regular、Slate 400、右下
区切り線: 下部 1pt、左インデント 80pt
```

**いいね済み表示**:
- 曲名の右にハートアイコン（小）を表示

### 検索機能

検索アイコンタップで検索バー展開:
```
┌─────────────────────────────┐
│ 🔍 曲名、アーティスト、ユーザー │
└─────────────────────────────┘
```

- リアルタイムフィルタリング
- 検索対象: 曲名、アーティスト名、ユーザー名

## セクション詳細

### ナビゲーションバー

- 左: 戻るボタン
- 右: メニュー（•••）→ 通報/ブロック/ミュート

### 曲情報セクション

**アルバムアート**:
```
サイズ: 200pt × 200pt
角丸: 16pt
シャドウ: 0 4pt 16pt rgba(0,0,0,0.15)
中央配置
```

**曲名・アーティスト**:
```
曲名: 24pt Bold、中央揃え
アーティスト: 17pt Regular、Slate 500、中央揃え
マージン: 上16pt
```

### アクションボタン

2つのボタンを横並び:

**再生ボタン**:
```
┌────────────────┐
│  ▶  再生する   │
└────────────────┘
```
- Spotify/Apple Musicで開く
- 未連携時: 連携促進ダイアログ

**いいねボタン**:
```
┌────────────────┐
│  ♡  いいね    │   ← 未いいね
└────────────────┘

┌────────────────┐
│  ♥  いいね済み │   ← いいね済み
└────────────────┘
```
- タップでトグル
- いいね時: ハートがバウンスアニメーション

**ボタンスタイル**:
```
高さ: 48pt
幅: 親要素の50% - 8pt
角丸: 12pt
背景: Slate 100（ライト）/ Slate 800（ダーク）
```

### すれ違い情報

```

今日 10:30 にすれ違いました
```

**スタイル**:
```
テキスト: 15pt Regular、中央揃え
```

**タップ**: ユーザープロフィール画面へ（将来実装）

### コメントセクション

**ヘッダー**:
```
コメント (3)
```
- フォント: 17pt Semibold
- 件数はバッジ的に表示

**コメントアイテム**:
```
┌───────────────────────────────────┐
│  ┌────┐  ユーザーA       1時間前  │
│  │ 👤 │                          │
│  └────┘  この曲いいですね！       │
└───────────────────────────────────┘
```

**スタイル**:
```
アバター: 32pt × 32pt、円形
ユーザー名: 13pt Semibold
時間: 13pt Regular、Slate 400
コメント: 15pt Regular
マージン: 下16pt
```

**自分のコメント**:
- 長押しで削除メニュー表示

### コメント入力

**レイアウト**:
```
┌─────────────────────────────────┐
│ 💬 コメントを入力...       送信  │
└─────────────────────────────────┘
```

**仕様**:
- 高さ: 52pt（最大120pt、複数行対応）
- 最大文字数: 200文字
- 送信ボタン: 入力があるときのみアクティブ
- キーボード表示時: 画面を押し上げ

**送信後**:
- コメントリストに即時追加
- 入力欄クリア
- 軽いフィードバックアニメーション

## アニメーション

### いいねタップ
1. ハートアイコンが一瞬拡大（1.2倍）
2. バウンスして元のサイズに
3. 色が変化（グレー → 赤系またはアクセント）

### コメント追加
1. 新規コメントが上からスライドイン
2. 軽いフェードイン

### 画面遷移
- 一覧→詳細: 右からスライドイン
- アルバムアート: シェアドエレメントトランジション（可能な場合）

---

## ダークモード

- アルバムアートのシャドウ: 無効化
- カード背景: `#1E293B`
- コメント入力欄: `#334155`

---

## アクセシビリティ

### VoiceOver/TalkBack

**詳細画面**:
- 「曲名、アーティスト名。再生ボタン。いいねボタン、未選択。」
- 「ユーザー名さんと今日10時30分にすれ違いました」

**コメント**:
- 「ユーザーAさんのコメント、1時間前。この曲いいですね！」

### 操作
- いいね: ダブルタップで切り替え
- コメント長押し削除: アクセシビリティアクションとしても提供
