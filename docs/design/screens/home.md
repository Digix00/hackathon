# ホーム画面 UI設計

## 概要

アプリのメイン画面。ユーザーが毎日最初に目にする画面として、「今日のすれ違い状況」と「シェア中の曲」を明快に表示する。

## セクション詳細

### 1. ナビゲーションバー

**構成**:
- 左: ハンバーガーメニュー（または非表示）
- 右: 通知アイコン + 設定アイコン

**仕様**:
```
高さ: 44pt
背景: 透明（スクロール時に背景色適用）
アイコンサイズ: 24pt
```

**通知バッジ**:
- 未読がある場合、通知アイコンに赤丸バッジ
- バッジサイズ: 8pt

---

### 2. 挨拶セクション

**目的**: パーソナルな体験の演出（Nightly参考）

**パターン**:
| 時間帯 | テキスト |
|---|---|
| 5:00-11:00 | おはようございます、〇〇さん |
| 11:00-17:00 | こんにちは、〇〇さん |
| 17:00-21:00 | こんばんは、〇〇さん |
| 21:00-5:00 | 夜遅くまでお疲れさまです |

**スタイル**:
```
フォント: 24pt Semibold
色: テキストプライマリ
マージン: 上24pt、下16pt、左右16pt
```

---

### 3. 一押しソングカード（ヒーロー表示）

**目的**: 現在シェア中の曲をジャケット中心に大きく見せる

**レイアウト**:
```
┌─────────────────────────────────┐
│  今日シェアしている曲            │  ← ラベル（13pt、Slate 500）
│                                 │
│       ┌─────────────┐           │
│       │             │           │
│       │  ジャケット  │           │  ← ヒーロー表示（120pt）
│       │             │           │
│       └─────────────┘           │
│                                 │
│           曲名                  │  ← 曲名（20pt Bold、中央揃え）
│        アーティスト名            │  ← アーティスト（15pt、中央揃え）
│                                 │
│          [変更]                 │  ← テキストボタン
└─────────────────────────────────┘
```

**カードスタイル**:
```
背景: カード背景色
角丸: 16pt
パディング: 24pt
シャドウ: 0 4pt 16pt rgba(0,0,0,0.1)
テキスト配置: 中央揃え
```

**アルバムアート（ヒーロー）**:
```
サイズ: 120pt × 120pt
角丸: 12pt
シャドウ: 0 8pt 24pt rgba(0,0,0,0.15)
マージン下: 16pt
```

**ドミナントカラー演出**（オプション）:
- ジャケットから抽出した Primary カラーで薄いグロー効果
- カード背景にグラデーション: Primary(5%) → 透明

**曲未設定時**:
```
┌─────────────────────────────────┐
│  今日シェアしている曲            │
│                                 │
│       ┌ ─ ─ ─ ─ ─ ─ ┐          │
│       │             │          │
│       │  + 🎵       │          │  ← 点線ボーダー、音符アイコン
│       │             │          │
│       └ ─ ─ ─ ─ ─ ─ ┘          │
│                                 │
│        曲を設定する              │
│                                 │
│  曲を設定すると、近くの人に       │
│  シェアされます                  │  ← Slate 500
└─────────────────────────────────┘
```

### 4. 今週出会った音楽（コラージュ）

**目的**: ジャケットを並べて「音楽との出会いの軌跡」を視覚的に表現

**レイアウト**:
```
今週出会った音楽                    すべて→

┌──┐┌──┐┌──┐┌──┐┌──┐
│  ││  ││  ││  ││  │   ← 各 56pt × 56pt
└──┘└──┘└──┘└──┘└──┘
┌──┐┌──┐┌────┐
│  ││  ││+12 │              ← 8枚目以降は「+N」バッジ
└──┘└──┘└────┘
```

**グリッド仕様**:
```
アイテムサイズ: 56pt × 56pt
角丸: 6pt
ギャップ: 8pt
1行あたり: 5枚
最大表示: 7枚 + 「+N」バッジ
```

**「+N」バッジ**:
```
背景: Slate 200（ライト）/ Slate 700（ダーク）
テキスト: 15pt Semibold、Slate 600
角丸: 6pt
```

**インタラクション**:
- 各ジャケットタップ → 該当のすれ違い詳細画面へ
- 長押し → クイックプレビュー（曲名 + アーティストのツールチップ）
- 「すべて→」タップ → すれ違い履歴一覧へ

**空状態**:
- セクション自体を非表示

**アニメーション**:
- 新しいすれ違い発生時、先頭にジャケットがスライドイン
- 他のジャケットは右にシフト

### 5. すれ違いサマリ

**目的**: 数値で活動状況を可視化

**レイアウト**:
```
  すれ違い

  ┌──────────────┐  ┌──────────────┐
  │     今日      │  │    今週      │
  │              │  │              │
  │      12      │  │      47      │
  │      人      │  │      人      │
  └──────────────┘  └──────────────┘
```

**カードスタイル**:
```
背景: カード背景色
角丸: 12pt
パディング: 16pt
幅: 親要素の50% - 8pt（ギャップ16pt）
```

**数値スタイル**:
```
数値: 32pt Bold
単位: 13pt Regular、Slate 500
```

**ゼロ状態**:
- 数値「0」を表示
- 補足テキスト「街を歩いて出会いを見つけよう」

---

### 6. 最近の出会いリスト

**目的**: 直近のすれ違いをクイックアクセス

**ヘッダー**:
```
最近の出会い                    すべて→
```
- 「すべて→」タップで履歴一覧画面へ

**リストアイテム**:
```
┌─────────────────────────────────────┐
│  ┌────┐                            │
│  │ 🎵 │  曲名              ユーザー名 │
│  │    │  アーティスト名       3分前  │
│  └────┘                            │
└─────────────────────────────────────┘
```

**アイテムスタイル**:
```
高さ: 72pt
アルバムアート: 48pt × 48pt、角丸 6pt
曲名: 15pt Semibold
アーティスト: 13pt Regular、Slate 500
ユーザー名: 13pt Regular
時間: 13pt Regular、Slate 400
```

**表示件数**: 最大 5 件

**空状態**:
```
┌─────────────────────────────────────┐
│                                     │
│     まだすれ違いがありません         │
│                                     │
│     街を歩いて、新しい音楽と         │
│     出会いましょう                   │
│                                     │
└─────────────────────────────────────┘
```

### 7. タブバー

**構成**:
| 位置 | アイコン | ラベル | 遷移先 |
|---|---|---|---|
| 1 | house.fill | ホーム | 現在画面 |
| 2 | list.bullet | 履歴 | すれ違い履歴一覧 |
| 3 | heart | お気に入り | いいね一覧 |
| 4 | person.circle | プロフィール | プロフィール/設定 |

**スタイル**:
```
高さ: 83pt（Safe Area含む）
背景: 背景色 + 上部ボーダー 1pt
アイコンサイズ: 24pt
ラベル: 10pt
アクティブ: アクセントカラー
非アクティブ: Slate 400
```

## インタラクション

### Pull to Refresh
- 画面を下に引っ張ってデータ更新
- ローディングインジケーター: Slateカラーのスピナー

### リストアイテムタップ
- すれ違い詳細画面へ遷移
- タップ時: 背景が薄くハイライト

### 曲変更ボタン
- 曲選択モーダルを表示
- 選択後、カードがフェードで更新

## リアルタイム更新

### すれ違い発生時
1. 画面上部からトースト通知スライドイン
2. すれ違い数カウントアップアニメーション
3. 最近の出会いリストに新規アイテム追加（上部に挿入）

**トースト（ジャケット付き）**:
```
┌───────────────────────────────────┐
│  ┌────┐                          │
│  │ 🎵 │  新しいすれ違い！         │
│  │    │  曲名 - アーティスト      │
│  └────┘                          │
└───────────────────────────────────┘

ジャケットサイズ: 40pt × 40pt
角丸: 6pt
背景: blur(20pt) + カード背景色 90%
```
- 表示時間: 3秒
- タップで詳細画面へ
- 上からスライドイン、フェードアウト

## ダークモード

- 背景: `#0F172A`
- カード: `#1E293B`
- シャドウ: 無効化（ボーダーで代替）
- アルバムアート: 周囲に薄いボーダー追加

---

## アクセシビリティ

### VoiceOver/TalkBack
- 挨拶セクション: 「こんにちは、〇〇さん」
- すれ違いカード: 「今日のすれ違い、12人」
- リストアイテム: 「曲名、アーティスト名、ユーザー名さんと3分前にすれ違い」

### ダイナミックタイプ
- 最大200%まで対応
- 大サイズ時はリストアイテムを縦積みに変更
