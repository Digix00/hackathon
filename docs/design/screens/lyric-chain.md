# 歌詞チェーン UI設計

## コンセプト

**「一言が、曲になる」**

すれ違いの瞬間に感じたことを、歌詞の一節として残す。
見知らぬ人々の言葉が集まり、一つの曲が生まれる体験。

---

## 画面一覧

| 画面 | 説明 |
|------|------|
| 歌詞入力モーダル | すれ違い成立後の歌詞投稿 |
| チェーン進捗表示 | 現在のチェーン状態を表示 |
| 生成完了通知 | 楽曲生成完了時のフルスクリーン演出 |
| 生成曲一覧 | 自分が参加した楽曲リスト |
| 生成曲詳細 | 楽曲再生・参加者表示 |

---

# 1. 歌詞入力モーダル

## 表示タイミング

すれ違いカード表示後、「聴く」or「いいね」の後に表示。
スキップ可能。

## レイアウト

```
┌─────────────────────────────────────┐
│                              ╳      │  ← 閉じるボタン
│                                     │
│     ✨ この出会いに一言             │
│                                     │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │  今日も空は青かった          │   │  ← テキスト入力
│  │                             │   │
│  │                      32/100 │   │  ← 文字数カウント
│  └─────────────────────────────┘   │
│                                     │
│     この歌詞は他のすれ違いと        │
│     組み合わさり、AIが              │
│     曲を作ります                    │
│                                     │
│  ┌─────────────────────────────┐   │
│  │        歌詞を残す            │   │  ← プライマリボタン
│  └─────────────────────────────┘   │
│                                     │
│         スキップ                    │  ← テキストリンク
│                                     │
└─────────────────────────────────────┘
```

## スタイル

| 要素 | 仕様 |
|------|------|
| モーダル背景 | 白（ライト）/ Slate 800（ダーク） |
| 角丸 | 24pt（上部のみ）|
| タイトル | 20pt Bold、中央揃え |
| テキストエリア | 高さ 120pt、Slate 50 背景、角丸 12pt |
| 文字数カウント | 13pt、Slate 400 |
| 説明文 | 14pt、Slate 500、中央揃え |
| ボタン | 高さ 52pt、幅 100%、Slate 900 背景 |

## インタラクション

| アクション | 動作 |
|-----------|------|
| 入力開始 | キーボード表示、モーダル上スライド |
| 100文字超過 | 入力不可、カウント赤色 |
| 空入力で送信 | ボタン非活性 |
| 「歌詞を残す」タップ | API送信 → 成功表示 → 閉じる |
| 「スキップ」タップ | モーダル閉じる |
| 背景タップ | モーダル閉じる |
| ╳ タップ | モーダル閉じる |

## 送信成功表示

```
┌─────────────────────────────────────┐
│                                     │
│              ✨                     │
│                                     │
│      歌詞を残しました！             │
│                                     │
│   あと 2人 で曲が生まれます         │
│                                     │
│         [チェーンを見る]            │
│                                     │
└─────────────────────────────────────┘
```

- 1.5秒後に自動で閉じる
- 「チェーンを見る」で詳細画面へ遷移

# 2. チェーン進捗表示

## ホーム画面での表示

参加中のチェーンがある場合、ホーム画面に進捗カードを表示。

```
┌─────────────────────────────────────┐
│                                     │
│  🎵 あなたの歌詞が曲になる途中      │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  ○ ─ ○ ─ ◉ ─ ○            │   │  ← 進捗ドット
│  │  3/4 人参加中               │   │
│  │                             │   │
│  │  「今日も空は青かった」      │   │  ← 自分の歌詞
│  │                      [見る] │   │
│  └─────────────────────────────┘   │
│                                     │
└─────────────────────────────────────┘
```

## 進捗ドットのスタイル

```
○ = 未参加スロット（Slate 300）
◉ = 自分の歌詞（Slate 900、塗りつぶし）
● = 他の人の歌詞（Slate 600）

例: 4人中3人参加（自分は2番目）
○ ─ ◉ ─ ● ─ ○
```

## 演出仕様

| 要素 | 仕様 |
|------|------|

| 音符アイコン | 64pt、ゆっくり浮遊アニメーション |
| 曲タイトル | 28pt Bold |
| 参加人数 | 16pt、Slate 400 |
| ボタン | 白背景、黒文字 |
| 星エフェクト | パーティクル、ゆっくり落下 |
| ハプティクス | 成功パターン（長め） |
